# Data-Professional-Survey-Dashboard
# 📊 Data Professional Survey Dashboard

An interactive Power BI dashboard created to analyze survey responses from data professionals and uncover trends across salaries, job roles, work satisfaction, preferred programming languages, and career experiences.

This project demonstrates data cleaning, transformation, modeling, and dashboard design using Excel and Power BI.

## Project Objective

The objective of this project is to transform raw survey data into meaningful business insights through interactive visualizations.

This dashboard helps answer questions such as:

- Which data roles earn the highest salaries?
- Which programming languages are most preferred?
- How satisfied are professionals with work-life balance?
- Which countries contribute most survey responses?
- How difficult is it to enter the data industry?

## Data Preparation

The following preprocessing steps were performed:

- Removed unnecessary columns
- Handled missing values
- Cleaned text fields
- Standardized categories
- Prepared data model in Power BI

## Key Insights

- Data-related roles showed noticeable salary differences.
- Programming language preferences varied across professionals.
- Work-life balance influenced overall satisfaction.
- Career entry difficulty differed by experience and role.
